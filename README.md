# 🧠 Depression Lifestyle Predictor

Application de data storytelling analysant le lien entre les habitudes 
de vie et le risque de dépression.

## Problématique
Les habitudes quotidiennes (sommeil, activité physique, stress, 
utilisation du téléphone...) peuvent-elles permettre de prédire 
un risque de dépression ?

## Stack technique
- Python, Pandas, NumPy
- Scikit-learn
- Plotly, Seaborn, Matplotlib
- Streamlit

## Installation
```bash
git clone https://github.com/erine2007/depression-lifestyle-predictor.git
cd depression-lifestyle-predictor
pip install -r requirements.txt
streamlit run app.py
```

## Structure du projet
- `data/` :  datasets bruts et traités
- `notebooks/` : analyse exploratoire
- `src/` : fonctions Python
- `pages/` : pages Streamlit
- `docs/` : documentation technique
